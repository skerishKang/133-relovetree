# Archive 관리 정책

이 디렉토리는 Lovetree 프로젝트의 레거시 코드, 프로토타입, 폐기된 UI 자산을 보관하는 구역입니다.

## 분류 체계

1. **recovered-legacy**: 구조 정리 중 복구된 과거 소스 코드 및 패킷. (보관 유지)
2. **prototype**: 기능 검토를 위해 작성된 프로토타입 및 개념 증명용 코드. (참조용 유지)
3. **old-ui**: V2 개편으로 인해 더 이상 사용하지 않는 과거 디자인 및 페이지. (나중 삭제 가능)
4. **reference-only**: 코드는 아니지만 기술적 참조를 위해 남겨둔 문서나 자산. (참조용 유지)

## 운영 수칙

- 프로젝트 루트가 지저분해지는 것을 방지하기 위해 정기적으로 이 구역으로 이동합니다.
- 운영 코드(src, pages, index.html 등)는 이 디렉토리 내부의 파일을 참조해서는 안 됩니다.
- 12개월 이상 지난 `old-ui` 자산은 팀 검토 후 삭제할 수 있습니다.
