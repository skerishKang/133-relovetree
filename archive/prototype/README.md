# Prototype

이 폴더는 Lovetree 개발 과정에서 진행된 각종 개념 증명(PoC) 및 프로토타입 코드를 보관합니다.

## 포함 내용
- `simple-prototype`: 초기 UI 구상 및 스케치
- `prototype-snapshots-*`: 특정 시점의 테스트 스냅샷
- `frontend-concept-v2-*`: V2 개편 초기 컨셉 코드

## 보관 상태
- **상태**: 참조용 유지 (Reference Only)
- **이유**: 향후 새로운 기능 제안이나 UI 개편 시 아이디어 뱅크로 활용
- **삭제 지침**: 아이디어가 현재 운영 코드에 완전히 흡수되었거나 더 이상 가치가 없다고 판단될 때 삭제.
