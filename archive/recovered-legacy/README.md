# Recovered Legacy

이 폴더는 과거 구조 정리 과정에서 루트(Root) 혹은 주요 디렉토리에서 분리된 레거시 소스 코드를 보관합니다.

## 포함 내용
- `legacy-entry-*`: 과거 진입점 스택
- `source-packets-*`: 이전 버전의 소스 묶음
- `root-cleanup-*`: 시스템 정리 중 분리된 기타 파일

## 보관 상태
- **상태**: 보관 유지 (Persistent Archive)
- **이유**: 과거 로직 확인 및 롤백 참조용
- **삭제 지침**: 보관 주기는 반영구적이며, 소스 제어 시스템(Git)에 충분히 기록된 경우에만 삭제 고려.
