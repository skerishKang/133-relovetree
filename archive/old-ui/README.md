# Old UI

이 폴더는 Lovetree 개편(V2) 이전의 HTML, CSS 및 관련 UI 자산을 보관합니다.

## 포함 내용
- `legacy-ui-*`: 구버전 CSS 및 디자인 자료
- `pages-old-*`: 폐기된 구버전 HTML 페이지

## 보관 상태
- **상태**: 나중 삭제 후보 (Candidate for Deletion)
- **이유**: 현재 디자인과 호환되지 않으나, 마이그레이션이 덜 된 텍스트 콘텐츠 확인용으로 일시 보존
- **삭제 지침**: 컨셉과 텍스트가 모두 V2에 반영된 것을 확인한 후, 최장 6개월 이내에 정기 삭제 권장.
