Lovetree concept interview

러브트리는
어떤 앱이어야 할까?

정답을 고르는 설문이 아니라, 실제로 쓸 사람의 감각을 듣기 위한 짧은 인터뷰입니다. 편하게 보고 “이건 좋다 / 이건 애매하다 / 나는 다르게 쓸 것 같다”를 남겨 주세요.

1. 정체성이 앱이 무엇처럼 느껴지는지 봅니다.
2. 기능기록, 연결, 영상 저장 방식이 이해되는지 봅니다.
3. 화면 방향어떤 화면이 가장 쓰고 싶은지 봅니다.

Part 1. Identity

처음 들었을 때, “러브트리”는 뭐 하는 앱처럼 느껴져?

가장 가까운 느낌을 골라줘. 여러 개 골라도 됩니다.

이 앱이 하나만 잘해야 한다면, 뭐가 제일 중요할까?

Lovetree의 중심축을 고르는 질문입니다.

너라면 첫 러브트리로 뭘 만들 것 같아?

실제 사용 시나리오를 확인하려고 합니다.

Part 2. Function

“순간 하나”를 저장할 때 꼭 들어가야 하는 건?

너무 복잡하면 안 쓰게 되고, 너무 단순하면 차별성이 없어집니다.

영상은 어떻게 저장되는 게 제일 자연스러워?

처음에는 YouTube가 가장 현실적이고, TikTok/Instagram은 이후 확장 후보입니다.

러브트리는 “연결된 트리”여야 할까, “시간순 기록”이면 충분할까?

이 질문이 제일 중요합니다. 서비스 차별성을 정하는 부분입니다.

Part 3. Screens

아래 화면 중 “이게 러브트리 같다”는 건 뭐야?

예쁜 것보다, 실제로 이해되고 쓰고 싶은 화면을 골라줘.

“순간 추가하기” 화면은 이해돼?

링크를 붙이고, 감정 태그와 메모를 남기고, 어느 순간과 연결할지 고르는 흐름입니다.

Part 4. Final

이 앱을 한 문장으로 소개한다면?

사용자 입장에서 이해되는 말인지 확인하고 싶습니다.

마지막으로, “이거 있으면 써볼 것 같다” 싶은 기능이 있어?

작은 기능이어도 좋습니다. 실제 사용 이유가 될 수 있습니다.

답변 전달 방법

버튼을 누르면 답변이 정리됩니다. 카톡이나 메일로 보내려면 “답변 복사”를 누르면 되고, 파일로 남기려면 “텍스트로 저장”을 누르면 됩니다.

아직 정리된 답변이 없습니다.
