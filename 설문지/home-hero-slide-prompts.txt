Lovetree Home Hero Slide Prompts

목적
- 홈 히어로 오른쪽 시각 영역을 “트리가 자라나는 과정” 슬라이드 5장으로 구성하기 위한 이미지 생성 프롬프트 모음
- 먼저 이미지 5장을 만들고, 이후 홈 슬라이드 또는 짧은 영상 소스로 사용

사용 원칙
- 화면 비율: 데스크톱 홈 히어로에 맞는 가로형 비율 권장 (16:10 또는 16:9)
- 톤: warm cream, dusty rose, muted sage, soft brown
- 공통 무드: K-pop 팬의 감정 기록, 영상 카드, 사랑의 흐름, 러브트리 성장
- 공통 구조: 왼쪽에서 시작해 오른쪽으로 성장하는 트리 흐름
- 실제 BTS 얼굴 재현보다 “현재 BTS 활동기 무드의 공연/뮤비/직캠 감성”에 가깝게

프롬프트 버전 구분
- SAFE VERSION: 실존 인물 직접 재현을 피하고 K-pop 보이그룹 무드로 생성
- BTS MOOD VERSION: BTS를 연상시키는 현재 무드, 공연 에너지, 팬 기억 톤을 더 강하게 반영


Slide 01

화면 카피
- 첫 순간이 러브트리의 시작이 됩니다

추가 요소
- 빈 배경 위에 트리의 메인 줄기와 5개 노드 자리만 은은하게 보이는 상태
- 아직 카드 내용은 거의 없음

SAFE VERSION
Create a premium desktop hero visual for a K-pop fan memory service called Lovetree.
Show a warm cream canvas with a soft organic tree structure starting from the left and growing toward the right.
Only the main trunk and five subtle node placeholders are visible, as if the memory tree is about to begin.
Use dusty rose, muted sage, ivory, and warm brown tones.
The visual should feel airy, emotional, and editorial, not technical.
No real celebrities, no copyrighted thumbnails, no logos.
The node placeholders should look like elegant video memory cards waiting to be filled.

BTS MOOD VERSION
Create a premium desktop hero visual for a K-pop fan memory service called Lovetree.
The scene should feel inspired by a global K-pop boy group fandom journey in the current era: stadium lights, emotional performance memories, fan-cam energy, warm concert afterglow.
Show an elegant tree-like path beginning on the left and extending to the right, with five subtle empty card placeholders.
No exact celebrity face recreation, no copyrighted logos, but the mood should evoke the emotional scale of BTS-era fandom.
Use ivory, dusty rose, muted sage, soft brown, and faint stage-light gradients.
