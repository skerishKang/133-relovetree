# Lovetree Image Generation Prompts

이 문서는 Lovetree 설문/프론트 리디자인용 화면 이미지를 생성하기 위한 프롬프트 모음입니다.

공통 규칙:
- 실제 아이돌 사진, 실제 플랫폼 로고, 저작권 이미지 사용 금지
- 영상 썸네일은 추상적 블러 이미지 또는 일러스트형 placeholder로 표현
- 컬러 방향은 warm cream, soft rose, muted green, muted slate
- 분위기는 감정적 팬 다이어리, 영상 아카이브, 부드러운 러브트리
- n8n, Figma, 개발자용 플로우차트처럼 보이지 않게 하기

---

## 1. `home-landing.png`

```text
Create a warm, premium landing page mockup for Lovetree, a K-pop fan memory app.

Concept:
Lovetree helps fans save meaningful moments as video cards and connect them into a growing LoveTree.

Screen:
- Desktop web landing page
- Warm cream background with soft rose, muted green, and slate accents
- Hero title in Korean: “사랑에 빠진 모든 순간을 기록해 보세요”
- Subtitle: “영상, 날짜, 감정, 메모가 이어져 나만의 러브트리가 됩니다”
- Primary CTA button: “첫 러브트리 만들기”
- Secondary CTA button: “다른 러브트리 구경하기”
- Show a preview of a connected LoveTree with 4 to 6 video memory cards
- Include small sections for “기록하기”, “연결하기”, “다시 보기”, “공유하기”
- No real idol photos, no real brand logos, no copyrighted content
- Use abstract video thumbnails only
- Emotional, soft, modern, fan diary mood
```
